内置函数

由python提供的函数功能, 就是内置函数, 常见内置函数有以下几种. 本节示例都是和匿名函数连用

max

去一个可迭代对象的最大值 , 是按照 ASCII 码的先后顺序进行比较.
参数:
max(可迭代对象, key= 比较对象)

示例1

1

4

l = [2, 3, 6, 8, 1, 10]

res = max(l)

print(res)

dict1 = {
    'tank': 1000,
    'egon':/ 500,
    'sean': 200,
    'jason': 500
}

res= max(dict1, key = lambda x:dict1[x])
print(res)

dict2 = sorted(dict1.items(), lambda x:x[1])

l = [1, 2, 3, 4]

s = map(lambda x:x+2, l)
print(s)

<!--用list 取出值-->

print(list(s)
# [3, 4, 5, 6]

reduce 合并

  • 参数1 函数对象
